# smart-events-backend
Owned by SLP

## Install locally
* `bundle install`
* `touch .env`
* Add secrets to .env

## API Routes
### Events
`/api/events`
- GET, POST

`/api/events/{id}`
- GET, PUT, DELETE

`/api/events/{id}/attractions`
- GET

### Attraction
`/api/attractions`
- GET, POST

`/api/attractions/{id}`
- GET, PUT, DELETE

### Attendee
`/api/attendees`
- GET, POST

`/api/attendees/{id}`
- GET, PUT, DELETE
