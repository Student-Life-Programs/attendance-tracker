# smart-events-backend
Owned by SLP

## Install locally
* `bundle install`
* `touch .env`
* Add secrets to .env

## API Routes
### Events
`/api/events`
- GET, POST

`/api/events/{id}`
- GET, PUT, DELETE

### Attractions
`/api/attractions`
- GET, POST

`/api/attractions/{id}`
- GET, PUT, DELETE

### Engagements
`/api/engagements`
- GET, POST

`/api/engagements/{id}`
- GET, PUT, DELETE

`/api/engagements/{id}/engagees`
- GET

### Engagees
`/api/engagees`
- GET, POST

`/api/engagees/{id}`
- GET, PUT, DELETE
